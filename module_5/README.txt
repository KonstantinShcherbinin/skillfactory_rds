Проект по созданию модели предсказания цены автомобиля по набору характеристик.
В ходе проекта пришлось:
1. Написать парсер для подтягивания данных с сайта auto.ru. С этим я справился, но заняло очень много времени на написание и отладку кода и сам парсер получился очень медленным.
2. Провести преварительный анализ данных. С этим справлися более-менее хорошо, но были ограничения по набору данных, полученных после парсинга.
3. Создать модель регрессии. Использовал три модели: RandomForest, Catboost, XGBoost.

Результат на kaggle получился посредственным (на момент написания 71 место)
Что можно улучшить:
1. Парсер. Широкое поле для развития. Нужно скоращать время его работы и число данных для сбора.
2. При анализе данных создать дополнительные признаки по комплектации автомобилей. Думаю это даст сущесственный рост точности предсказания.
3. Попробовать ансамблирование и блендинг моделей.
